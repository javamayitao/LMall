Mall

项目简介

主要是采用springcloud技术栈来进行服务拆分和治理，提高系统的并发能力。含有完整的前端页面，从用户选择商品-加入购物车-生成订单-支付宝扫码支付-后台管理系统。大体实现所有功能。除了springcloud技术栈，还涉及分布式锁、MQ异步下单、redis预减库存等内容
